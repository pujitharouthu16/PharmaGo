# Pharma Management System

## Overview

The Pharma Management System is a simple application designed to manage medicines, stock, and sales in a pharmacy. It helps pharmacists organize medicine records and track inventory efficiently.

## Features

* Add new medicines
* Update medicine details
* Delete medicine records
* Search medicines
* Manage stock inventory
* View available medicines

## Technologies Used

* HTML
* CSS
* JavaScript

## Usage

* Add medicine information.
* Update stock when medicines are sold or restocked.
* Search for medicines by name.
* View inventory details.

## Future Enhancements

* Online medicine ordering
* Billing system
* User authentication
* Sales reports
